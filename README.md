# HMCTS Task Management System



A full-stack task management application for HMCTS caseworkers.



## Structure



- `backend/` — Node.js + Express REST API with SQLite

- `frontend/` — React application



## Quick Start



### Backend

```bash

cd backend

npm install

npm start

```



### Frontend

```bash

cd frontend

npm install

npm start

```



See `backend/README.md` and `frontend/README.md` for full documentation.
